{{ _('Creative Brief') }}

{{ _('Call to Action') }}

{% trans %} We need your help to tell the amazing story of Mozilla Firefox, our users and how one browser can build a brighter future for the Web. {% endtrans %}

{{ _('Project Description') }}

{% trans %} Create a video ad or story (a "flick") in one of four categories that helps people better understand the issues facing them online and shows how Firefox helps them deal with these issues. In short, help us tell the story of Firefox — why we do what we do and how that helps users on the Web. {% endtrans %}

{{ _('Project Summary') }}

{% trans %} Mozilla wants the world to know more about what makes Firefox different: that we're a non-profit organization aimed at keeping the power of the Web in people's hands. Our hope is that through high-quality videos, we can help mainstream Internet users know more about the issues that affect their online lives and, more importantly, how Firefox addresses these issues. Winning videos will be used in worldwide marketing campaigns and public service announcements. {% endtrans %}

{{ _('Objective') }}

{% trans %} Help us educate consumers about one or all of the following online issues facing them today: {% endtrans %}

  1. {% trans %} Privacy: How are you being tracked online? How can you take control of it? Who owns your data and how do they use it? {% endtrans %}
  2. {% trans %} Choice: How easy is it for you to experience the Web using the tools that you like and that suit your needs? Can you make it personal? {% endtrans %}
  3. {% trans %} Interoperability: Can you use your favorite tools on all your devices, different computers or in different locations? Who decides when, where and how you use the Web and access your stuff? {% endtrans %}
  4. {% trans %} Opportunity: What tools do users need to help them make the Web they want? How does Firefox help? {% endtrans %}

{% trans brand='http://www.mozilla.org/firefox/brand/platform/' %} Also, drive awareness of our key brand attributes, specifically around the fact that Firefox is made by a non-profit organization that answers only to users, not shareholders. Have a look at our Brand Platform for more details and check out the video below. {% endtrans %}

{{ _('Who are we Talking To?') }}

{% trans %} Your flick should target mainstream Internet users between the ages of 25 and 54 — people who use the Web to read the news, write e-mails, bank online and connect on social networks. Help explain the challenges facing the Web in terms that these average everyday users will understand. As an extreme example, how would you explain the importance of having control over your own online experience to someone who only understands the Firefox logo as the "Internet" button? We need you to communicate to this audience that Firefox is a non-profit doing good for the benefit of people everywhere. {% endtrans %}

{{ _('Primary Key Message') }}

{% trans %} Firefox prioritizes principles over profit to put you in control of your online experience and shape the future of the Web for the public good. Firefox answers to no one but you. {% endtrans %}

{{ _('Secondary Key Message Points:') }}

  1. {% trans %} There are important online issues like privacy, choice, flexibility and opportunity facing you every day. {% endtrans %}
  2. {% trans %} Mozilla addresses these issues and works to ensure that the Web remains open and accessible to all. {% endtrans %}
  3. {% trans %} Mozilla is proudly non-profit. {% endtrans %}

{{ _('Call to Action') }}

{% trans %} Tell the world about Firefox. Encourage them to download Firefox today. {% endtrans %}

  1. {% trans brand='http://www.mozilla.org/firefox/brand/' %} Firefox logo: Please see our Brand Toolkit details on how to use our logo {% endtrans %}
  2. {% trans %} Call to action {% endtrans %}

{{ _('What do we want the Viewer to Know and Feel?') }}

{{ _('Know:') }}

  1. {% trans %} We are proudly non-profit and dedicated to doing good. {% endtrans %}
  2. {% trans %} We are competing with some of the world's biggest companies to put you in control of your online experience. {% endtrans %}
  3. {% trans %} Firefox is developed together with a global community in the open, driven by what's best for users everywhere. {% endtrans %}
  4. {% trans %} We prioritize principles over profits. {% endtrans %}

{{ _('Feel:') }}

  1. {% trans %} Your video should be more grassroots than corporate to reflect that Mozilla is a public benefit organization and not a for profit company. {% endtrans %}
  2. {% trans %} It should inspire the viewer to be a part of the Mozilla Firefox movement. {% endtrans %}
  3. {% trans %} It should help spread the word about what makes us different and how downloading Firefox makes the Web and world a better place. {% endtrans %}

{{ _('Tone') }}

{% trans %} Unconventional, Engaging, Honest, Human, Smart, Confident {% endtrans %}

{{ _('Guidelines:') }}

  1. {% trans %} Take risks, but respect the brand (i.e. please don't make us look bad). {% endtrans %}
  2. {% trans %} Your flick should function at the brand level, not the product level (you don't need to talk about product features — talk about how Firefox and the movement around it make you feel). {% endtrans %}
  3. {% trans %} It can be fun and humorous, but keep it engaging. {% endtrans %}
  4. {% trans %} All that said, these guidelines are just that: guidelines. Be creative, think outside the box and, most importantly, have fun. {% endtrans %}
  5. {% trans brand='http://www.mozilla.org/firefox/brand/' %} Take a look at the Mozilla Firefox Brand Toolkit to see how we talk about ourselves. {% endtrans %}
